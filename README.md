## 🎯 게임 목록

### 구현 완료 게임
1. **테트리스** - 클래식 블록 퍼즐 게임

5. **스도쿠** - 9x9 숫자 퍼즐 게임

8. **게임 도우미 챗봇** - AI 대화형 게임 안내

## 🚀 설치 및 실행

### Visual Studio에서 실행
1. [Project3.sln](Project3.sln) 파일을 Visual Studio로 열기
2. 빌드 구성을 Debug 또는 Release로 설정
3. `F5` 키를 눌러 빌드 및 실행

### 명령줄에서 빌드
```bash
# Visual Studio Developer Command Prompt에서
msbuild Project3.sln /p:Configuration=Release
```

### 실행 파일 직접 실행
```bash
# Debug 폴더에서
.\TEAM10.exe
```

## 🎮 게임 상세 설명

### 1. 테트리스
- **조작법**: 방향키로 이동/회전, 스페이스바로 빠른 낙하
- **목표**: 가로줄을 완성하여 높은 점수 획득
- **특징**: 클래식한 테트리스 규칙 구현

### 2. 지뢰찾기
- **조작법**: 좌표 입력으로 칸 열기 (`r 3 4`), 깃발 표시 (`f 1 2`)
- **목표**: 모든 지뢰를 피해 안전한 칸 모두 열기
- **특징**: 첫 클릭은 항상 안전, 연쇄 열기 기능

### 3. 미로 탐험
- **조작법**: WASD로 이동, Z로 뒤로가기
- **목표**: P(플레이어)에서 E(출구)까지 도달
- **특징**: 3가지 난이도, 실시간 타이머

### 4. 스네이크
- **조작법**: 방향키로 뱀 이동
- **목표**: 먹이를 먹으며 뱀을 성장시키기
- **특징**: 레벨별 장애물 증가, 속도 조절

### 5. 스도쿠
- **조작법**: 행, 열, 숫자 순서로 입력
- **목표**: 9x9 격자를 1-9 숫자로 완성
- **특징**: 난이도 선택, 시간 제한, 자동 풀이 기능

### 6. 행맨
- **조작법**: 알파벳 입력으로 단어 추측
- **목표**: 행맨이 완성되기 전에 단어 맞추기
- **특징**: 영어 학습 효과, 시간 제한

### 7. 우주선 게임
- **조작법**: 방향키로 이동, 스페이스바로 폭탄 사용
- **목표**: 운석을 피하고 아이템 수집
- **특징**: 체력 시스템, 특수 아이템

### 8. 게임 도우미 챗봇
- **기능**: 자연어 처리를 통한 게임 추천
- **특징**: AI 대화형 모드, 게임 정보 제공
